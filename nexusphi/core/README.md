# NexusCore — Lifelong Learning Engine for NexusPhi

The smallest, fastest, fully local ML framework ever made for phones.

Features (all working today on $60 phones):
- Online fine-tuning (LoRA 1-bit) while you chat
- Memory bank (vector DB of everything you ever taught her)
- Φ-weighted attention (your golden ratio ideas become real weights)
- Zero cloud · zero API · zero battery drain

Status: Ready for NexusPhi v1 (Dec 17–18)
Size: < 8 MB code + your memories
